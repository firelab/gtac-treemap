This readme contains descriptions of the files and folders accompanying the attribute dataset, {outputFileName}_{col_name}.
{additional_toptext}


Files included:

	{outputFileName}_{col_name}.tif - The raster dataset. Values from the corresponding attribute {col_name} of the main {outputFileName} dataset have been
				       written to this raster's pixel values.

	{outputFileName}_{col_name}.html - The raster dataset's html metadata, viewable via a web browser.

	{outputFileName}_{col_name}.xml - The raster dataset's xml metadata, viewable via a web browser.

	{outputFileName}_{col_name}.tif.xml - The raster dataset's ArcGIS compatible metadata, viewable in ArcGIS Pro.

	{outputFileName}_{col_name}.tif.aux.xml - The raster dataset's ArcGIS compatible statistics, used by ArcGIS Pro for visualization and analysis.
