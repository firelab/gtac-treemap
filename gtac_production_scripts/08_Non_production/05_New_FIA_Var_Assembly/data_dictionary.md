# Data Dictionary: TreeMap Volume Attributes

This document defines the output variables generated by the standalone variable assembly workflow.

## Volume Attributes

| Variable Name | Description | Units | Calculation Logic (FIA Query Filters) |
| :--- | :--- | :--- | :--- |
| **VOLCFNET_L** | Net cubic-foot volume, Live trees | ft³/acre | Sum of `VOLCFNET * TPA_UNADJ` where:<br>- `COND_STATUS_CD = 1` (Forest)<br>- `STATUSCD = 1` (Live tree) |
| **VOLCFNET_D** | Net cubic-foot volume, Standing Dead trees | ft³/acre | Sum of `VOLCFNET * TPA_UNADJ` where:<br>- `COND_STATUS_CD = 1` (Forest)<br>- `STATUSCD = 2` (Dead tree)<br>- `DIA >= 5` (Diameter $\ge$ 5 inches)<br>- `STANDING_DEAD_CD = 1` (Standing dead) |
| **VOLCFSND_L** | Sound cubic-foot volume, Live trees | ft³/acre | Sum of `VOLCFSND * TPA_UNADJ` where:<br>- `COND_STATUS_CD = 1` (Forest)<br>- `STATUSCD = 1` (Live tree) |
| **VOLCFSND_D** | Sound cubic-foot volume, Standing Dead trees | ft³/acre | Sum of `VOLCFSND * TPA_UNADJ` where:<br>- `COND_STATUS_CD = 1` (Forest)<br>- `STATUSCD = 2` (Dead tree)<br>- `STANDING_DEAD_CD = 1` (Standing dead) |

## Pixel-Level Attributes (Raster Outputs)

These variables are derived from the per-acre attributes above, converted to represent the total volume within a single 30x30m pixel.

| Variable Name | Description | Units | Calculation Logic |
| :--- | :--- | :--- | :--- |
| **VOLCFSND_L_PX** | Sound cubic-foot volume, Live trees (Per Pixel) | ft³/pixel | `VOLCFSND_L` * (900 m² * 0.000247104 acres/m²) |
| **VOLCFSND_D_PX** | Sound cubic-foot volume, Standing Dead trees (Per Pixel) | ft³/pixel | `VOLCFSND_D` * (900 m² * 0.000247104 acres/m²) |

## Identifiers

| Variable Name | Description |
| :--- | :--- |
| **CN** (or PLT_CN) | Unique identifier for the FIA plot condition. |
| **TM_ID** | Unique identifier for the TreeMap imputation class. |
